The ``1-my_list.txt`` module
============================

This library has a function that print sorted list called
print_sorted.

Firstly, initialize the module
::
        >>> MyList = __import__('1-my_list').MyList

Now use it
::
        >>> my_list = MyList()

check append
::
        >>> my_list.append(1)
        >>> my_list.append(4)
        >>> my_list.append(2)
        >>> my_list.append(3)
        >>> my_list.append(5)

To sort a list
::
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

check non-append
::
	>>> my_list = MyList([3, 1, 2, 4, 5])
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

check non-appended negative number
::
	>>> my_list = MyList([3, -1, -2, 4, 5])
	>>> my_list.print_sorted()
        [-2, -1, 3, 4, 5]

check empty list
::
	>>> my_list = MyList([])
	>>> my_list.print_sorted()
	[]

check str
::
	>>> my_list = MyList([3, -1, -2, 4, 5])
	>>> print(my_list)
	[3, -1, -2, 4, 5]
